# Size Tracker - MVP 1

Android-приложение для отслеживания веса и калорий.

## Функционал

### Реализованные возможности ✅

- **Онбординг**: Ввод начальных данных пользователя (вес, рост, возраст, пол)
- **Автоматический расчет калорий**: По формуле Миффлина-Сан Жеора
- **Отслеживание веса**: Запись и просмотр истории изменений веса
- **Отслеживание калорий**: Учет потребленных калорий за день
- **Главный экран**: Отображение текущего прогресса
- **История**: Просмотр всех записей веса и калорий

## Технологии

- **Язык**: Kotlin
- **UI**: Jetpack Compose + Material 3
- **База данных**: Room Database
- **Архитектура**: MVVM
- **Навигация**: Jetpack Navigation Compose
- **Минимальная версия Android**: 7.0 (API 24)

## База данных

### Таблицы

1. **user_profile**: Профиль пользователя
2. **weight_entries**: Записи веса
3. **calorie_entries**: Записи калорий

## Формула расчета калорий

Используется формула Миффлина-Сан Жеора:

**Для мужчин:**
```
BMR = 10 × вес(кг) + 6.25 × рост(см) - 5 × возраст(г) + 5
```

**Для женщин:**
```
BMR = 10 × вес(кг) + 6.25 × рост(см) - 5 × возраст(г) - 161
```

Итоговая норма = BMR × коэффициент активности (по умолчанию 1.2)

## Критерии готовности MVP 1

- ✅ Можно ввести вес и сохранить
- ✅ Можно ввести калории и сохранить
- ✅ Видно итог калорий за день

## Экраны

1. **Онбординг**: Первый запуск, ввод данных пользователя
2. **Главный экран**: Текущий вес, калории за день, кнопки действий
3. **Добавление веса**: Ввод нового значения веса
4. **Добавление калорий**: Ввод потребленных калорий
5. **История**: Просмотр всех записей с возможностью удаления
